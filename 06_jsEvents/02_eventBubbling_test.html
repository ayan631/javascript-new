1. Event Bubbling:

Event Bubbling works from bottom to top elements. If the 3rd parameter of 'addEventListener()' function is set to 'false' for both child and its parent nodes, then events will work as Bubbling, which means the child event will be executed first and then the parent event will be executed.


2. Event Capturing:

Event Capturing works from top to bottom elements. If the 3rd parameter of 'addEventListener()' function is set to 'true' for both child and its parent nodes, then events will work as Capturing, which means parent event will be executed first, then the child event will be executed.
